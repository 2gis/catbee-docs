Catbee.
Isomorphic framework based on «Single State Tree» conception. Catbee use Baobab as main data storage and web componets as view layer. Framework preaches unidirectional data flow, and split logic and side effects. This ideas highly inspired by Cerebral.
Catbee have builtin blazing fast server render engine and efficient client-side rendering based on morphdom library.
